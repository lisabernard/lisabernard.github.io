My Create Your Own Story Game

Since October 2016, I have been developing a make your own story game based on a series called Invader Zim. It started out as a basic python program. Click here to see my code

I later figured programming in python was a dead end, so I later switched to html. Every decision a user can make is a separate web page, as opposed to an if statement. I recently completed a demo version of the game which is compatable with all browsers and android mobile devices!
